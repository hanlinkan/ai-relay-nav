# AI中转站导航

数据同步自 [aibijia.org](https://aibijia.org)，实时比价帮你找便宜Token。

## 功能

- 📊 ChatGPT、Claude、Gemini、Grok 全平台比价
- 🔍 搜索、筛选、排序
- 📈 最低价、平均价、最高价统计
- 🔗 直接跳转购买链接

## 数据来源

- [aibijia.org](https://aibijia.org) - AI订阅一键比价平台
- 数据API: `https://data.aibijia.org/products.json`

## 技术栈

- Vue 3 + Vite
- TailwindCSS
- 部署: Vercel
