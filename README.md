# 🤖 AI中转站导航

对比不同AI API中转站的价格、口碑、速度，帮你找到最合适的AI服务。

## ✨ 功能

- 📊 **价格对比** - 同模型不同中转站价格一目了然
- ⭐ **用户评价** - 真实口碑和评分
- ⚡ **速度测试** - API响应速度对比
- 🔍 **智能筛选** - 按模型、地区、价格筛选
- 📱 **响应式设计** - 手机电脑都能用

## 🚀 技术栈

- Vue 3 + Vite
- TailwindCSS
- 响应式设计
